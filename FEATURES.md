* ✔ **Soporte para resoluciones adaptativas personalizadas**

  El juego permite configurar una resolución interna de renderizado adaptativa, respetando automáticamente la relación de aspecto de cada dispositivo Android.

  Para obtener más información sobre esta funcionalidad y aprender cómo cambiar la resolución, revisa el archivo:

* ✔ **Limitador de FPS añadido a 60 frames por segundo**

  Como ocurre con muchos juegos de principios de los **años 2000**, el motor del juego utiliza la conocida variable **delta time**.

  Si el juego se ejecuta a una tasa superior a **60 FPS**, pueden aparecer problemas como:

  * físicas incorrectas
  * comportamientos inesperados en el gameplay
  * errores en animaciones o lógica del juego

  Por este motivo se ha añadido **un límite de 60 FPS** para mantener la experiencia original.

---

* ✔ **VSync añadido**

  El **VSync** sincroniza los **FPS generados por la GPU** con la **frecuencia de refresco del monitor o pantalla**, evitando problemas visuales como:

  * **screen tearing**
  * cortes horizontales en la imagen

---

* ✔ **Soporte para mandos externos**

  El juego es compatible con **controladores externos**, utilizando un esquema de control adaptado a los **estándares modernos de mandos**.

  Se ha incorporado **vibración para mandos**, solo en **64 bits**, ya que muchos dispositivos **32 bits** tienen problemas para reconocer, por sus propios drivers, si un mando puede vibrar o no.

---

* ✔ **Gatillos analógicos mejorados para conducción**

  Los gatillos **L2 / R2** o **LT / RT**, dependiendo del mando, ahora funcionan de forma **progresiva** en los vehículos.

  Esto permite:

  * una **aceleración más precisa**
  * **mejor control de la velocidad**
  * una experiencia de conducción **más natural y cómoda**

## Features Latest Version

* ✔ **Minimap moved** when using touch controls. [**Touch controls**](TOUCH_CONTROLS.md)

---

* ✔ **Story mode fully playable from start to finish**, with no issues during gameplay.
  **Cutscenes play correctly** during the game. See **[KNOWN_ISSUES.md](KNOWN_ISSUES.md)** for more details about possible minor issues.

---

* ✔ **Fully functional audio playback**, maintaining the original game behavior.

---

* ✔ **Excellent performance on the vast majority of modern Android devices**
  approximately **year 2020 or newer**.

  On more modest devices, the game will **still be 100% comfortably playable**, although performance may vary depending on the hardware.

---

* ✔ **Support for custom adaptive resolutions**

  The game allows you to configure an adaptive internal render resolution, automatically preserving the aspect ratio of each Android device.

  For more information about this feature and to learn how to change the resolution, check the file:

  [How to install](INSTALLATION.md)

---

* ✔ **FPS limiter added at 60 frames per second**

  As with many games from the early **2000s**, the game engine uses the well-known **delta time** variable.

  If the game runs above **60 FPS**, issues may appear such as:

  * incorrect physics
  * unexpected gameplay behavior
  * animation or game logic errors

  For this reason, a **60 FPS cap** has been added to preserve the original experience.

---

* ✔ **VSync added**

  **VSync** synchronizes the **FPS generated by the GPU** with the **monitor or screen refresh rate**, preventing visual issues such as:

  * **screen tearing**
  * horizontal image artifacts

---

* ✔ **Support for external controllers**

  The game is compatible with **external controllers**, using a control scheme adapted to **modern controller standards**.

  **Controller vibration support** has been added, only on **64-bit**, since many **32-bit** devices have issues recognizing, due to their own drivers, whether a controller can vibrate or not.

---

* ✔ **Improved analog triggers for driving**

  The **L2 / R2** triggers, or **LT / RT** depending on the controller, now work in a **progressive** way in vehicles.

  This allows:

  * more **precise acceleration**
  * **better speed control**
  * a more **natural and comfortable** driving experience

  [Installationsanleitung](INSTALLATION.md)

* ✔ **FPS-Begrenzung auf 60 Bilder pro Sekunde hinzugefügt**

  Wie bei vielen Spielen aus den frühen **2000er Jahren** verwendet die Spiel-Engine die bekannte Variable **delta time**.

  Wenn das Spiel mit mehr als **60 FPS** läuft, können Probleme auftreten, wie zum Beispiel:

  * fehlerhafte Physik
  * unerwartetes Verhalten im Gameplay
  * Fehler bei Animationen oder in der Spiellogik

  * **Screen Tearing**
  * horizontale Bildfehler

  Das Spiel ist kompatibel mit **externen Controllern** und verwendet ein Steuerungsschema, das an **moderne Controller-Standards** angepasst ist.

---

* ✔ **Verbesserte analoge Trigger für die Fahrzeugsteuerung**

  Die Trigger **L2 / R2**, oder **LT / RT** je nach Controller, funktionieren nun **progressiv** in Fahrzeugen.

  Dies ermöglicht:

  * eine **präzisere Beschleunigung**
  * eine **bessere Geschwindigkeitskontrolle**
  * ein **natürlicheres und komfortableres** Fahrerlebnis
